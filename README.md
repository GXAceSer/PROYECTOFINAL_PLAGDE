# PLAGDE -  PLATAFORMA GENERADORA DE DOCUMENTOS ESCOLARES 
Proyecto final de la materia **Sistemas distribuidos** Semestre 2025-2

## Descripción
PLAGDE es una plataforma web que permite generar documentos escolares como constancias, diplomas y citatorios de manera rápida, segura y con identidad institucional.

## Objetivo
Facilitar la creación de documentos oficiales cumpliendo con normas del sector educativo, reduciendo tiempo y errores manuales.

## Características 
- Genreración automática de documentos.
- Personalización de información.
- Descarga de archivos formato PDF.
- Interfaz fácil de usar.

## Tecnologías usadas:
### Backend
-Python 
-Flask

### Frontend
-HTML
-CSS

### Base de datos
-PostgreSQL

### Infraestructura
-Nginx
-Docker Compose
